Planar Reflections 2
Motivation
  • Fix the problems we had last time with planar reflections
Recall
  • We have two problems with our scene:
    • Things that are behind the mirror are visible “through” it
      • Ex: The cylinder
    • Reflections can “poke out” from behind the mirror
      • Ex: The cone
Fix
  • We can solve the first problem like so:
  • Render the mirror to the depth buffer only (i.e., mask off color buffer)
  • Then draw the non-reflected objects as usual
    • Depth buffer will ensure objects that fall behind mirror are not rendered
  • Then draw the reflected scene
  • Finally, draw the mirror with α < 1
Code
void draw(){
    glClear( GL_COLOR_BUFFER_BIT | GL_DEPTH_BUFFER_BIT );
    Program::setUniform("reflectionMatrix", mat4::identity() );
    glColorMask(false,false,false,false);
    drawMirror();
    glColorMask(true,true,true,true);
    drawObjects();
    Program::setUniform("reflectionMatrix", R );
    drawObjects();
    Program::setUniform("reflectionMatrix", mat4::identity() );
    drawMirror();
}
Results
  • That worked a little too well
Why
  • The depth buffer culls away both the reflected objects and the objects behind the mirror
  • Solution: Clear the depth buffer just before drawing reflected objects

Results
  • We still have problems
  • The mirror renders on top of all other objects
    • Not surprising, since we wiped out the depth buffer
  • Reflections can still poke out from behind the mirror
  • If reflected geometry intersects the mirror, the depth test might reject the mirror fragments
    • Which is happening near the cube’s corner
Solution
  • We can fix this problem with the stencil buffer
  • At setup time: Enable it:
    glEnable(GL_STENCIL_TEST);
Draw
  • When we start rendering the frame, clear all three buffers:
    • glClear(GL_COLOR_BUFFER_BIT | GL_DEPTH_BUFFER_BIT | GL_STENCIL_BUFFER_BIT );
  • Here’s what the color, depth, and stencil buffers would contain:
Draw
  • Next, we want to set the stencil buffer to one’s where the mirror is
  • Then we draw the mirror to the depth buffer only
    • glStencilFunc(GL_ALWAYS,1,0xff);
    • glStencilOp(GL_KEEP,GL_KEEP,GL_REPLACE);
Code

    • glStencilFunc(GL_ALWAYS,0,0xff);

Draw
  • Only permit rendering where stencil buffer is 1:
    • glStencilFunc(GL_EQUAL,1,0xff);
    • glStencilOp(GL_KEEP,GL_KEEP,GL_KEEP);

Result
Draw
  • Disable the depth test: glDisable(GL_DEPTH_TEST)
  • Draw the mirror itself
    • Stencil test is still GL_EQUAL with op’s == GL_KEEP
      • We can only render where stencil buffer is 1
      • Stencil buffer will not be modified at all
    • End result: Stencil test prevents drawing where mirror is covered by other objects
    • But mirror can cover anything in the reflection, since depth test not active
Code
void draw(){
    glClear( GL_COLOR_BUFFER_BIT | GL_DEPTH_BUFFER_BIT | GL_STENCIL_BUFFER_BIT );
    Program::setUniform("reflectionMatrix", mat4::identity() );
    glStencilFunc(GL_ALWAYS,1,0xff);
    glStencilOp(GL_KEEP,GL_KEEP,GL_REPLACE);
    glColorMask(false,false,false,false);
    drawMirror();
    glColorMask(true,true,true,true);
    glStencilFunc(GL_ALWAYS,0,0xff);
    drawObjects();
    Program::setUniform("reflectionMatrix", R );
    glClear(GL_DEPTH_BUFFER_BIT);
    glStencilFunc(GL_EQUAL,1,0xff);
    glStencilOp(GL_KEEP,GL_KEEP,GL_KEEP);
    drawObjects();
    glDisable(GL_DEPTH_TEST);           //new
    Program::setUniform("reflectionMatrix", mat4::identity() );
    drawMirror();
    glEnable(GL_DEPTH_TEST);
}
Result
  • Perfect!
Note
  • Sometimes, it doesn’t look good if the reflections are too sharp
Blurry
  • We can use FBO’s to help us get blurred reflections
  • Procedure:
    • Create an FBO
    • At render time:
      • Make FBO render target
      • Do all of the previous steps as usual, but stop just before the final drawMirror()
        • Note: Render into FBO
Blurry
  • With FBO still bound, call
    globs->fbo.blur(0,0,11.0,1.0f);
    • Note: Will need to use new version of Framebuffer.h that handles stencil test correctly
  • Result: FBO is blurred only where mirror is located
    • Because stencil test is still set up for that
Blurry
  • Then finish the render:
drawMirror();                   //just draw the mirror
globs->fbo.unsetAsRenderTarget();
glEnable(GL_DEPTH_TEST);
glClear(GL_COLOR_BUFFER_BIT | GL_DEPTH_BUFFER_BIT | GL_STENCIL_BUFFER_BIT);
glStencilFunc(GL_ALWAYS,1,0xff);
globs->fboProg.use();       //simple prog: Just render texture to output
globs->fbo.texture->bind(0);
globs->fullscreenQuad.draw();
Note
